### init ver.

- 支持 TOTP 以及 HOTP
- 支持手动输入参数，扫码，解析 `otpauth://` 三种导入方式
- 支持动态取色
- 支持动态语言切换
- 支持webdav云备份和回复
- 自定义拖拽item
- 指纹验证登录
